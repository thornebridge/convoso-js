DATA SOURCES
================================================================================

This documentation was built by merging two authoritative sources:

1. HELP CENTER EXTRACTION (62 articles)
   - Source: https://help.convoso.com/hc/en-us/categories/360002169153-API
   - 62 articles extracted across 18 categories via Chrome browser automation
   - Provided: use cases, workflow examples, code samples, tutorials

2. ADMIN API REFERENCE (internal documentation)
   - Provided directly from the platform's Admin view
   - 16 API sections covering all 37 endpoints
   - Provided: authoritative URLs, error codes, field specifications
   - Treated as the canonical source for all technical details

Where the two sources conflicted, the Admin API reference was used as the
authoritative source. Key corrections from the Admin reference:
   - ALL endpoints use POST method (Help Center incorrectly showed some as GET)
   - Several URL corrections (e.g., /lists/ plural, /campaigns/status plural)
   - Comprehensive error codes added for every endpoint
   - Additional fields and parameters documented
